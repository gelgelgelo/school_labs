# 1st year college school labs  

## Installation / Cloning the repo  

the default branch only contains documents of a lab which allows you to  
do the labs by yourself and learn too.  

```  
git clone https://github.com/gelgelgelo/school_labs.git  
``` 

alternatively, you can review my code by switching to the `devel` branch.  

```  
git checkout devel  
```
